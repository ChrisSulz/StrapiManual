An abstraction around `react-query`'s `useQuery` hook. It can be used to fetch permissions for one admin role.

## Usage

The hooks can receive two optional parameters:

1. query params: an object containing the query params to be sent to the API. They are going to be
   stringified by `qs`. All params are equal except `id`, which is used to fetch a single users, if
   it is passed.
2. options: an object containing the options to be passed to `useQuery`.

It returns an object containing some of the react-query attributes.

## Typescript

```ts
import { UseQueryOptions } from 'react-query'

type Permissions = object;

useAdminRolePermissions(queryParams: object, reactQueryOptions: UseQueryOptions): {
    permissions: Permissions,
    isLoading: boolean;
    error: object;
    isError: boolean;
    refetch: () => Promise<void>;
};
```

### Fetch all permissions for a role

```jsx
import { useAdminRolePermissions } from 'path/to/hooks';

const MyComponent = ({ onMoveItem }) => {
  const { permissions, isLoading, refetch } = useAdminRolePermissions({ id: roleId });

  return /* ... */;
};
```
